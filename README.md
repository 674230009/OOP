# Object-Oriented Programming (OOP) - PHP

รวบรวมเนื้อหาและตัวอย่างการเขียนโปรแกรมเชิงวัตถุ (Object-Oriented Programming) ด้วยภาษา PHP

## หัวข้อการเรียนรู้ (Topics)

- [x] **Abstract Classes** - คลาสแม่แบบที่ไม่สามารถนำไปสร้างวัตถุได้โดยตรง
- [x] **Access Modifiers** - การกำหนดระดับการเข้าถึง (`public`, `protected`, `private`)
- [x] **Classes & Objects** - การสร้างคลาสและการเรียกใช้งานวัตถุ
- [x] **Constants** - การกำหนดค่าคงที่ในคลาส
- [x] **Constructor & Destructor** - เมธอดพิเศษสำหรับสร้างและทำลายวัตถุ
- [x] **Inheritance** - การสืบทอดคุณสมบัติระหว่างคลาส
- [x] **Interfaces** - การกำหนดสัญญากรรมวิธีสำหรับคลาส
- [x] **Iterables** - การใช้งานข้อมูลที่สามารถวนลูปได้
- [x] **Namespaces** - การจัดกลุ่มคลาสเพื่อป้องกันชื่อซ้ำ
- [x] **Static Methods & Properties** - การใช้งานสมาชิกของคลาสโดยไม่ต้องสร้างวัตถุ
- [x] **Traits** - การแชร์โค้ดระหว่างคลาสต่างสายเลือด
